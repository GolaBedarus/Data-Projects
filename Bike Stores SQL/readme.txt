Este proyecto contiene un "dummy" dataset, el cual sirve como punto de apoyo para mostrar JOINS y Funciones agregadas. 

DATASET: Bike Stores

Este proyecto de SQL se centra en analizar datos de ventas de una base de datos relacional para obtener información sobre el comportamiento de los clientes, el rendimiento de las tiendas, las ventas de productos y la eficacia de los representantes de ventas. Mediante la ejecución de una serie de consultas SQL complejas en múltiples tablas, el proyecto agrega métricas clave, como el total de unidades vendidas y los ingresos totales, y las vincula con los detalles de clientes, tiendas, productos y personal.
